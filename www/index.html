Description

The purpose of SwingSet3 is to provide a new showcase demo for Swing that highlights how the toolkit enables creation of state-of-the-art GUIs.
Goals: Non-Goals:
  • Demonstrate more complete component set
  • Show off toolkit feaures added since 1.3
  • Demonstrate common GUI configurations
  • Make it easy to view code for a particular feature
  • Make it easy to add demos
  • Show how a Swing GUI can be beautiful
  • Twiddle-every-property test application
  • Replacement for all prior swing demos
