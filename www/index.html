The purpose of SwingSet3 is to provide a new showcase demo for Swing that highlights how the toolkit enables creation of state-of-the-art GUIs. Specifically:

  • Show off the new Nimbus look-and-feel in Java SE 6 Update 10
  • Demonstrate toolkit feaures added since 1.3 (when SwingSet2 was written)
  • Make it easy to view code for a particular feature
  • Make it easy to add demos

SwingSet3 is NOT intended to be a twiddle-every-property test application nor a replacement for all prior Swing demos.

Although by default SwingSet3 comes up with a set of Swing core demos, the demo set could easily be replaced or augmented with other demos that have visual GUI components (Java2D, etc.). See Writing Demos for SwingSet3.

SwingSet3 requires Java 6 and will only display the Nimbus look-and-feel if run with Java 6 update10 release.
