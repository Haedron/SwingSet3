The purpose of SwingSet3 is to provide a new showcase demo for Swing that highlights how the toolkit enables creation of state-of-the-art GUIs. Specifically:

  • Demonstrate more complete component set; see target demo list.
  • Show off toolkit feaures added since 1.3 (when SwingSet2 was written)
  • Demonstrate common GUI configurations
  • Make it easy to view code for a particular feature
  • Make it easy to add demos
  • Show how a Swing GUI can be beautiful

SwingSet3 is NOT intended to be a twiddle-every-property test application nor a replacement for all prior Swing demos.

Although by default SwingSet3 comes up with a set of Swing core demos, the demo set could easily be replaced or augmented with other demos that have a visual GUI components (Java2D, etc.). See Writing Demos for SwingSet3 (it's easy!)
